## Module Management

### Via Gateway Webpage
1. Navigate to: Config → Modules
2. View installed modules
3. Install new modules (.modl files)
4. Enable/disable modules
5. Restart required after changes

### Via File System
1. Place .modl file in `user-lib\modules\`
2. Update `data\modules.json`
3. Restart Gateway

### Module States
- **Enabled:** Active and running
- **Disabled:** Installed but not active
- **Faulted:** Error during startup
